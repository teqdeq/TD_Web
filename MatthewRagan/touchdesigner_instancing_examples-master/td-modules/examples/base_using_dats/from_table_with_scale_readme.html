Using DATs For Instancing

Instances from a Table with Scale

Understanding our table DAT as a set of columns for data, where each row is an instance we can expand our set of examples to also include using the idea of scale. In this example we add three more columns, 'sx', 'sy', and 'sz'. While we only change the 'sy' value, you can experiment with changing other values in the table to see how that would impact our instances.
